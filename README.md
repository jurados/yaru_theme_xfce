### Console color

General
- **Texto color:** #F7F4DB
- **Background color:** #300A24
- **Tab Active:** #0F4999

Custom colors: Only select `Text selection color` and `Bold text color`
- **Text selection color:** #163B59 and no other.
- **Bold text color:** #FFFFFF
